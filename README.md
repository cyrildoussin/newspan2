# Newspan

With the demise of Google Reader, everyone is making RSS readers that suck a little. We are not the exception.

 * self-hosted RSS reader
 * labels, stars as expected
 * unlike some feed readers, we honour `xml:base`
 * keyboard shortcuts
 * Heroku compatible
